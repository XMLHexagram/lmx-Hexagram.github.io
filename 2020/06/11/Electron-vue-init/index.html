Vue CLI Plugin Electron Builder

想折腾一下electron + vue毕竟下一个项目要用到
然后，查electron在vue怎么配置

先是毫无参考价值的build之后的文件套壳，然后是vue-electron这种上一次更新在去年的库。。。

最后终于是找到了 Vue CLI Plugin Electron Builder

放上GitHub链接和文档链接

看看最后一次提交时间 14hours

使用方法:(官方文档)

Then, install and invoke the generator of vue-cli-plugin-electron-builder by running:
vue add electron-builder
That’s It! You’re ready to go!

有操作的啊 这个库

具体起步使用在文档的get start里都有

接下来最重要的是

vue add electron-builder之后，它做了什么

打开tower跟踪文件变更

一下文件被改变了

  • .gitignore
  • package.json
  • src
    • add background.js
  • yarn.lock(这个是yarn校验文件，不用管)

逐个分析

.gitignore

  • 添加了 忽略了build electron之后的文件(应该的)

package.json

“scripts”字段

"scripts"字段下新增了这四个命令
前两个好理解分别是 build(启动构建) serve(启动服务)
至于后两个的作用嘛 引用electron-builder文档的原话

add “main” 字段

main字段指定了electron的入口文件,这里指向了后面出现的background.js

“devDependencies” 字段

添加了两个开发依赖，看一下依赖的名字就知道干了什么吧

src/background.js

这个就是electron的基础配置文件，制定了一些最基础的行为

That’s all.
